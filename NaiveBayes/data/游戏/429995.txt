CJGameLab总裁：被收购是员工的意愿
       前身是韩国hofIsland，目前也是全新更名为CJ Game Lab公司。 这家公司曾经开发制作过《Hounds》，《Kingdoms》，《ProjectRaw》等多款网络游戏。
　　■CJ Game Lab公司简介
　　2009年4月成立至今已经有2年时间，原来公司名称叫hofIsland，但是公司在2010年11月被韩国CJ E&M公司收购后更名为CJ Game Lab名称，目前公司已经有200余名工作人员，那么在韩国国内也称的上是颇具实力的游戏公司之一。
　　而且现在正开发制作的游戏分别有TPS类型《Hounds》，战略网页游戏《Kingdoms》，FPS射击游戏《ProjectRaw》，SF MMORPG《ProjectK》等多款网络游戏作品预计都将在明年与玩家朋友见面。
　　“所有工作人员都是非常喜欢网络游戏”
　　虽然公司规模还不算很大，起码来说已经有了开发制作网络游戏经验也算是具有实力的工作团队。
　　以下让我们专访CJ Game Lab总裁郑韵相先生采访内容：
　　Q：首先请您简单介绍一下公司概况。
　　A：我们公司目前有200余名员工，而且分别有开发制作5款网络游戏作品。
　　同时，公司很多员工都有着丰富的网络游戏开发经历。 可以说他们都是有经验和实力的精英。
　　另外，我们公司员工都痴迷于网络游戏，应该说非常痴迷吧。 所以我们希望能够开发制作优秀的网络游戏作品提供给大众玩家们。
　　Q：公司前身是hofIsland，那么目前员工是否都是原班人马组成呢？
　　A:不是的。 虽然有些是原hofIsland员工，而且我们也招募了许多新员工。相对来说，这些新加入的员工也是自身能力非常强。目前员工累计起来大约有230名左右，预计未来会增加到300名。
　　Q：2010年被韩国CJ E&M公司收购原因是什么呢？
　　A:主要还是考虑到公司未来发展，而且公司部分员工也是希望能够成为CJ E&M公司一员。 
　　本身我们公司是比较注重员工想法，而且所有人都比较谦虚谨慎。实际在日常生活中大家都是互相帮助，同时作为管理层人员也是会倾听员工的一些建议内容。
　　那么选择成为CJ E&M公司也都是听从了员工们的意见后做出的决定。 随即公司名称也更改为CJ Game Lab也是员工的意见。
　　Q：CJ Game Lab公司目标是什么呢？
　　A：从长期发展角度来说，能够开发制作出具有创新内容，而且希望开发出的所有游戏都能够得到玩家们的关注和支持。还有公司曾经遇到过很多波折，不过在困难的时期也克服过来。所以未来通过这些员工的努力一定能够取得很大突破和成功。我们希望所有员工都能够幸福快乐的开发制作网络游戏。为此作为总裁会支持员工，为他们做好各项后勤保障工作。
