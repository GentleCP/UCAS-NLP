建筑大师伊东丰雄精心打造嘉茂园（组图）
　　新加坡，2010年9月3日：获奖无数的国际知名建筑师伊东丰雄，今天现身新加坡出席其最新作品嘉茂园(Belle Vue Residences)的发布会。嘉茂园(Belle Vue Residences)为伊东丰雄于日本以外的亚洲的首个住宅公寓项目。
　　嘉茂园(Belle Vue Residences)坐落于新加坡黄金地段第9区内，毗邻乌节路(Orchard Road)与欧思礼路(Oxley Road)，四周绿翠环抱，环境恬静，其建筑巧妙地与大自然和谐融合，为一项共设176个尊贵单位的永久业权豪华发展项目。
图为：嘉茂园全景
　　嘉茂园(Belle Vue Residences)占地约250,000平方?，广阔开扬，自然流畅的线条运用及富空间感的设计，与大自然生趣遥相呼应，充分展现了伊东丰雄著名的流线型几何设计风格。于此概念下，伊东丰雄从感性出发，将种种动态、韵律、空间转变、结合等特色巧妙揉合，视公寓建筑及四周环境为一个连续空间，从而流转出截然不同的细节形态。
图为：嘉茂园私人露天平台
　　伊东丰雄解释“为这发展项目打造亮丽的设计，我认为不需要刻意塑造形态。”他补充道：“当我看过嘉茂园(Belle Vue Residences)的选址后，便直觉地认为嘉茂园(Belle Vue Residences)的设计概念， 应与大自然的分枝形态相呼应，以突显此地的美丽和价值。我的设计意念为创造出一个让我们重新发现人与大自然应相互依存的居住环境，将户外的闲适恬静延伸至个人的生活空间之中。”
　　嘉茂园(Belle Vue Residences)的大自然有机形态布局是模拟树木的分支形态，创造出空间感，让每个单位均可享受煦日与凉风，以及各具特色的景观。此设计手法让更多外墙向外而立，有利充分采光及空气流通，将内外空间完美相连 。嘉茂园(Belle Vue Residences)的9栋公寓坐落于精心铺设的翠绿小径、热带园林及优美湖畔溪流之中，体现人性化设计，让人置身都会中心，依然能尊享难得的清幽写意。
图为：嘉茂园露天游泳池图片
　　嘉茂园(Belle Vue Residences)横跨于景观水池上的两层天桥式高会所，设观景阳台、多用途活动室、休憩廊及剧院。至于低层，则设有临优雅庭园花园而建的健身室及休憩前厅。
　　各公寓单位中由伊东丰雄所设计的浴室用品及室内设备系列，充分反映建筑师对细节的重视，其设计与生活空间互相呼应，突显嘉茂园(Belle Vue Residences)大自然气息洋溢的和谐感。
　　永泰控股副主席郑维荣先生表示：“我们很荣幸能与伊东丰雄合作，打造他日本以外，亚洲首项亦是唯一的一个住宅项目。伊东丰雄大师级的流线式几何设计与空间相配合，演绎了大自然恬适、随心的神韵。嘉茂园(Belle Vue Residences)建筑形态流畅自然，富有美感，为新加坡独一无二的物业项目。
图为：嘉茂园室内装修图片
　　郑先生补充道：“发展这种建筑风格别树一帜的项目，是我一直所期待的，我为这次与伊东丰雄先生难得的合作机会感到十分高兴。透过嘉茂园(Belle Vue Residences)，我们不但成功体现了该址的历史及传统韵味，伊东先生的建筑才华 更为空间赋予人性化的品味。在某种程度上说，我认为我们在演绎大自然精髓之余，更将都会生活概念提升至前所未有的境界。”
　　嘉茂园(Belle Vue Residences)为奢华生活的典范，三房公寓单位由1,600平方?起，四房公寓单位由2,000平方?起，可随时入伙，并计划于随后数星期内推出。
　　永泰简介
　　永泰控股有限公司以优越品质及设计著称，是新加坡顶尖的房地产发展商以及领导生活时尚的品牌，其主要业务为一家专注于亚洲增长市场的投资控股公司。永泰亚洲(Wing Tai Asia)的公司网络，包括永泰控股有限公司(新加坡)、永泰(中国)投资控股公司、DNP控股有限公司(马来西亚)、永泰地产有限公司及其各自的附属公司，其核心业务涵盖物业投资及发展、时尚消费品销售及酒店管理。
